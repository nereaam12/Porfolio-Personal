Proyecto 3

Proyecto académico desarrollado con Symfony cuyo objetivo es digitalizar la gestión de una biblioteca. La aplicación permite administrar libros, socios y préstamos mediante una interfaz clara, funcional y responsive.

Funcionalidades principales
  • Gestión completa del catálogo de libros
  • Gestión de socios con validaciones de datos
  • Control de préstamos y devoluciones
  • Listado de préstamos activos
  • Control automático de disponibilidad de libros
Tecnologías utilizadas
  • Symfony 7.4
  • MariaDB
Proceso de trabajo

Análisis de requisitos, diseño del modelo de datos, desarrollo backend con Symfony (entidades, relaciones, controladores y formularios) y validaciones personalizadas.

Aplicación de Gestión de Tareas

Proyecto académico enfocado al desarrollo de una aplicación web para la gestión de tareas personales. Permite organizar tareas, controlar su estado y gestionar la información mediante un backend en PHP y base de datos.

Funcionalidades principales
  • Creación, edición y eliminación de tareas
  • Marcado de tareas como completadas
  • Listado de tareas pendientes y finalizadas
  • Persistencia de datos en base de datos
Tecnologías utilizadas
  • PHP
  • MySQL
  • HTML5
  • CSS3
  • Bootstrap
Proceso de trabajo

Análisis de requisitos, diseño de la base de datos, desarrollo del backend en PHP y creación de una interfaz clara y responsive para el usuario.
